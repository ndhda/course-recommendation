About EduChoice

Read More

Overview

EduChoice is a web-based platform designed to help learners identify the best Coursera courses tailored to their preferences. By utilizing the Multi-Criteria Decision Making (MCDM) method, specifically the TOPSIS technique, EduChoice ranks courses based on user-selected criteria, ensuring personalized and relevant recommendations.

Project Goals:

  • Identify Course Selection Criteria: Understand the factors that influence user choices for online courses.
  • Develop a Recommendation System: Implement the TOPSIS method to provide data-driven course recommendations.
  • Ensure System Accuracy: Deliver precise and relevant course rankings to enhance the user experience.

Features

  • Personalized Recommendations: Users can customize criteria based on course difficulty, duration, and type.
  • User-Friendly Interface: Simple navigation with a clear 'GET RECOMMENDED' button for quick access.
  • Advanced Decision-Making: Utilizes the TOPSIS method for accurate course ranking.
  • Static Display: Information presented clearly, even when hosted on localhost.

How It Works

Course Criteria: Users select from categories such as:

  • Course Difficulty: Beginner, intermediate, mixed, or advanced.
  • Course Time: Options ranging from 1-4 weeks to 3-6 months.
  • Course Type: Including courses, specializations, and professional certificates.

Ranking System: The system processes user input through the TOPSIS method, generating a ranked list of courses.

Future Vision

While currently hosted on localhost, EduChoice envisions expanding to a broader audience, providing an accessible platform for learners worldwide. The goal is to continually improve the recommendation system, incorporating user feedback and expanding the range of supported platforms.
